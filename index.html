Moovies

Moovies is a free and opensource api that is used to do stuff about moovies.

Threre are a lot you can do with moovies.

You can :

  • Search for moovies with their titles.
  • Get the plot of the movie if there is one available.
  • Get the cast of the movie.
  • Get plot summary, plot synopsis.
  • Get reiews. [coming soon ... ]
  • Get ratings. [coming soon ... ]
  • and many more.....
